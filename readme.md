# C(clothes)Shop 미니 프로젝트

### 안녕하세요 웹뿌링클치킨이 아니라 웹뿌론트엔드가 되고싶은 삐약삐약 주니어치킨개발자 ABlued입니다!

html, css, 바닐라JS로 만든 미니 프로젝트 CShop에 대해 소개해드리겠습니다.

##### 중요 코드 로직

```
loadItems()
.then(items => {
    displayItems(items);
    setEventListeners(items)
})
.catch(console.log);
```
